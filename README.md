# SQL Analytics Challenge

## Objective
Perform advanced SQL analysis to understand sales performance, revenue trends, and customer behavior.

## Tools & Technologies
SQL (PostgreSQL/MySQL), Excel

## Methodology
1. Data extraction using complex SQL queries  
2. Aggregations and calculations for revenue, orders, and customer metrics  
3. Cohort analysis for retention  
4. Ranking top-performing products and customers

## Key Insights
- Best-selling products and categories  
- Monthly revenue trends  
- Top 10 customers by total spending

## Recommendations
- Prioritize stock for high-demand products  
- Design retention campaigns for loyal customers
